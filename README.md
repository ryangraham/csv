# CSV

A C++ header-only CSV parser with an interface inspired by Python's DictReader/DictWriter.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Install csv using [Homebrew](https://brew.sh/).

```
brew tap ryangraham/csv
brew install csv
```
